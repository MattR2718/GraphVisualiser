# GraphVisualiser

## Instructions
- Adjacency list is represented as a space separated list of strings<br>
    - [KEY][SPACE][ELEMENT1][SPACE][ELEMENT2]...<br>
- In program:
    - Key is a string
    - Elements are a vector of strings<br>
- In visualisation:
    - Arrow represented as WHITE &rarr; RED
